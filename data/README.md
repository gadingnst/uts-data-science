# Folder Data - Dataset UTS Data Science

Folder ini digunakan untuk menyimpan seluruh dataset yang dihasilkan oleh script scraper.

## Daftar Dataset
* `periplus_books_raw.csv` - Dataset mentah hasil scraping langsung dari Periplus.com (belum dibersihkan).
* `periplus_books_clean.csv` - Dataset bersih yang siap digunakan untuk analisis statistika deskriptif dan visualisasi data.

## Catatan
* File CSV di folder ini di-generate secara otomatis ketika Anda menjalankan script di folder `scrapers/`.
* Jangan mengubah isi file CSV secara manual agar hasil analisis tetap konsisten dengan data scraping.
